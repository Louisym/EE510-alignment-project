概率论基础知识

第一章：概率的基本概念

1.1 随机试验与样本空间
随机试验是指在相同条件下可以重复进行的试验，其结果在试验前不能完全确定。
样本空间（Sample Space）是随机试验所有可能结果的集合，通常用Ω表示。
样本点是样本空间中的一个元素，表示试验的一个可能结果。

1.2 事件
事件是样本空间的子集，表示试验的某种结果或某些结果的组合。
- 基本事件：只包含一个样本点的事件
- 必然事件：一定会发生的事件，即样本空间本身
- 不可能事件：不可能发生的事件，即空集

1.3 概率的定义
设Ω为样本空间，A为事件，则P(A)称为事件A的概率，满足：
1) 非负性：0 ≤ P(A) ≤ 1
2) 规范性：P(Ω) = 1
3) 可列可加性：对于两两互不相容的事件序列A₁, A₂, ...，有
   P(⋃ᵢ₌₁^∞ Aᵢ) = Σᵢ₌₁^∞ P(Aᵢ)

1.4 条件概率
条件概率是指在已知某个事件B发生的条件下，另一事件A发生的概率，记作P(A|B)。
定义：P(A|B) = P(A∩B) / P(B)，其中P(B) > 0

条件概率满足概率的所有性质。

1.5 独立性
两个事件A和B称为独立的，如果：
P(A∩B) = P(A)·P(B)

等价地，当P(B) > 0时，A和B独立等价于：
P(A|B) = P(A)

1.6 贝叶斯公式
设B₁, B₂, ..., Bₙ是样本空间的一个划分，且P(Bᵢ) > 0，则对任一事件A，有：
P(Bⱼ|A) = P(Bⱼ)P(A|Bⱼ) / Σᵢ₌₁ⁿ P(Bᵢ)P(A|Bᵢ)

这个公式在贝叶斯统计推断中有重要应用。

第二章：随机变量及其分布

2.1 随机变量
随机变量是定义在样本空间上的实值函数，通常用X, Y, Z等大写字母表示。
- 离散型随机变量：取值为有限个或可列无穷多个
- 连续型随机变量：取值为某个区间或整个实数轴

2.2 分布函数
随机变量X的分布函数定义为：
F(x) = P(X ≤ x)，x ∈ ℝ

分布函数具有以下性质：
1) 单调不减性
2) 右连续性
3) F(-∞) = 0，F(+∞) = 1

2.3 概率密度函数
对于连续型随机变量X，如果存在非负函数f(x)，使得对任意实数x，有：
F(x) = ∫_{-∞}^x f(t)dt

则称f(x)为X的概率密度函数。

2.4 常见分布
- 均匀分布：X ~ U(a,b)
- 指数分布：X ~ Exp(λ)，常用于描述等待时间
- 正态分布：X ~ N(μ, σ²)，最重要的连续分布
- 泊松分布：X ~ P(λ)，常用于描述单位时间内随机事件发生的次数

2.5 数学期望
离散型：E[X] = Σᵢ xᵢP(X = xᵢ)
连续型：E[X] = ∫_{-∞}^{+∞} xf(x)dx

期望的性质：
1) E[c] = c（c为常数）
2) E[aX + b] = aE[X] + b
3) E[X + Y] = E[X] + E[Y]
4) 若X和Y独立，则E[XY] = E[X]·E[Y]

2.6 方差
方差用于度量随机变量的离散程度：
Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²

标准差：σ = √Var(X)

第三章：多维随机变量

3.1 联合分布
对于二维随机变量(X,Y)，其联合分布函数为：
F(x,y) = P(X ≤ x, Y ≤ y)

3.2 边缘分布
从联合分布可以得到各分量的边缘分布：
Fₓ(x) = F(x, +∞)
Fᵧ(y) = F(+∞, y)

3.3 协方差与相关系数
协方差：Cov(X,Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]
相关系数：ρ = Cov(X,Y) / (σₓσᵧ)

相关系数的性质：
1) |ρ| ≤ 1
2) |ρ| = 1 当且仅当X和Y之间存在线性关系
3) ρ = 0时，称X和Y不相关（但不一定独立）
